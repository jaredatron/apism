# Apism

A simple framework, build on top of Sinatra, for building JSON APIs

## Installation

Add this line to your application's Gemfile:

    gem 'apism'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install apism

## Usage

apism new [APPLICATION_NAME]
